BLIT 这个名字缩写自 **B**anner**L**ord **I**mage **T**ool，我希望将它打造成一款处理骑砍 2 MOD 图片的全能工具，为各位 Modder 带来便利。❤️

这个应用还在开发阶段，将会提供更多功能。如果你发现了 bug 或者需要新功能，请在 [GitHub](https://github.com/0x1c-works/BLIT/issues) 上提交 issue 或 PR 来一起完善它！

<Callout emoji="😛">
  一开始没注意，后来发现起的这个名字恰好和著名的 `blit`
  函数重名了（在图形编程中，用这个函数将图像复制到另一个渲染缓冲区），不过转念一想，这不正好说明了这个应用和图像处理有亿点点关系吗！
</Callout>
