# 安装 BLIT

## 系统要求

- Windows 10 version 1809 (10.0.17763.0) 或更高版本
- Windows App Runtime 1.2
  > 应该会自动安装

---

## 安装

从[这里](https://github.com/0x1c-works/BLIT/releases)下载最新版本并解压到任意位置。

> BLIT 会安装一个由微软官方工具签名的证书到你的系统，对你的系统不会造成任何影响。不过因为我没钱买 CA 签发的证书，所以可能会有警告。

### Manually

1. Right click on the `.cer` file and "Install Certificate".
2. Run the `BLIT.Win_<VERSION>_x64.msix` file.
3. After installation, you should find BLIT in your start menu.

### With PowerShell

Run the `Install.ps1` in PowerShell with the following commands:

```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
./Install.ps1
Set-ExecutionPolicy -ExecutionPolicy AllSigned -Scope CurrentUser
```
